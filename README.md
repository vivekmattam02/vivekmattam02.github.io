My personal portfolio website showcasing my robotics projects, research experience, and technical skills.

## About

This portfolio highlights my work in robotics and autonomous systems, including:
- Data center automation robots at NYU
- Autonomous navigation and visual SLAM research
- Robotic manipulator control systems
- IoT and embedded systems projects

## Tech Stack

- React
- TypeScript
- SCSS
- GitHub Pages

## Development

To run locally:

```bash
npm install
npm start
```

To deploy:

```bash
npm run build
git checkout gh-pages
# Copy build files and commit
git push origin gh-pages
```
